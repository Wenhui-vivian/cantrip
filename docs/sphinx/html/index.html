reCurrent Additive Network for Temporal RIsk Prediction (CANTRIP)¶

A TensorFlow model for predicting temporal (disease) risk based on retrospective analysis of longitudinal clinical notes.

Please check the website for details.

Dependencies¶

  • Python >= 3.6
  • TensorFlow >= 1.9
  • SciPy
  • tqdm
  • tabulate

Installation¶

To install, run
$ python setup.py

Documentation¶

Documentation on CANTRIP is provided at here. Sphinx-based Python documentation is available here.

Structure¶

  • src/data Classes and utilities for loading clinical chronologies (and observation vocabularies from the disk); Python documentation is provided here
  • src/models TensorFlow implemenation of CANTRIP; Python documentation is provided here
    • src/models/encoder TensorFlow implementation of clinical snapshot encoders; Python documentation is provided here
    • src/models/rnn_cell TensorFlow implementation of Recurrent Additive Networks (RANs) and Batch-normalized Gated Recurrent Units; Python documentation is provided here
  • src/scripts Executable scripts for running and evaluating CANTRIP as well as SVM and other baseline systems on pneumonia risk predcition; Python documentation is provided here
