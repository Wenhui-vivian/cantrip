src.models.encoder.snapshot_encoder module¶

Clinical snapshot encoders for use with CANTRIP Model.

CANTRIPModel expects a clinical snapshot encoder function which takes as input the CANTRIPModel and adds clinical snapshot encoding ops to the graph returning the final clinical snapshot encoding as [batch x max_seq_len x embedding_size]
